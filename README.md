# HELLO!

This is a fun personal project. Web server and HTTP parser, written in cpp from scretch, multithreaded, directly utilizing the Linux socket API.
- why though?  
Because I want it to host my own website, where I will upload... I don't know what yet, probably CV, personal projects and a blog.

- still why though? You can just use node or flask or something like that.  
Because it's more fun to build it from scretch, I used node for it already, and it was braindead easy. Which is awesome that this technology
exists, but it's more fun to build your own for a small website that will probably not have too much traffic going through it.
